# Content Publish Orchestrator

## Role
MDX 콘텐츠 작성 및 퍼블리싱을 관리하는 오케스트레이터.

## Cycle

### 1. Research
- PRD의 Content Format 섹션 참조
- 글 구조: Problem -> System -> Implementation -> Result -> Insight
- SEO 키워드 확인

### 2. Execute
- skills/write-content.md 호출
- frontmatter 메타데이터 작성
- MDX 본문 작성
- 이미지/다이어그램 배치

### 3. Review
- MDX 파싱 정상 확인
- frontmatter 필수 필드 검증
- SEO 메타데이터 확인
- sot/review-log.md에 기록

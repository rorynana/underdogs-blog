첫 실행 시 `sot/writing-style-guide.md`로 저장하고, 이후에는 캐시된 가이드를 재사용한다.

## 실행 조건
- `sot/writing-style-guide.md`가 없을 때만 실행
- 이미 존재하면 이 스킬을 건너뛰고 기존 가이드 사용

## 학습 대상 파일

   - 시장 분석 (데이터 기반, 분석적 톤)

## 추출할 패턴

### 1. 문장 종결 패턴
- ~이다/~하다/~한다 (평서형)
- ~인데/~하는데 (연결형)
- 구어체 비율 vs 문어체
- 문장 길이 경향 (짧은/긴)

### 2. 자기 지칭
- "나는", "내가", "본인은", "우리는"
- 어떤 상황에서 어떤 지칭을 쓰는지

### 3. 감정/반응 표현
- 이모티콘 사용 여부와 패턴 (>_<, ㅠㅠ 등)
- 괄호 속 코멘트/사족 ((사실은...), *TMI* 등)
- 감탄/강조 표현 ("정말", "매우", "기가 막히게" 등)

### 4. 문단/섹션 구조
- 도입부 패턴 (바로 본론? 배경 설명?)
- 전환 표현 ("자 그럼", "이제", "다음으로" 등)
- 마무리 패턴 ("다음 편에서", "알아보자" 등)

### 5. 전문용어 처리
- 영문 병기 방식: "질화 철(nitrided steel)" vs "질화 철"
- 약어 사용: "ABO", "CBO" 등
- 괄호 설명 패턴

### 6. 유머/위트
- 어떤 상황에서 가벼운 톤을 쓰는지
- 자기비하 유머, 과장법, 비유 사용 패턴
- 밈/유행어 활용 방식

### 7. 독자 소통
- "여러분", "당신", 직접 호칭 없음 등
- 질문형 문장 사용 빈도
- "~하지 않는가?", "~해보자!" 등의 권유형

### 8. 이미지/미디어 활용
- 이미지 배치 빈도 (몇 문단마다?)
- 이미지 설명(alt text, 캡션) 스타일
- 이미지와 본문의 연결 방식

## Output

`sot/writing-style-guide.md` 파일 생성:

```markdown
# Writing Style Guide

## 톤앤매너 요약
(한 문단으로 전체 문체 특징 요약)

## 문장 패턴
(구체적 예시와 함께)

## 감정 표현
(사용 패턴과 예시)

## 구조 패턴
(도입-전개-마무리)

## 용어 처리
(영문 병기, 약어 등)

## 유머/위트
(사용 패턴과 예시)

## 미디어 활용
(배치 빈도와 방식)

## 참고 문장 샘플
(기존 글에서 추출한 특징적 문장 5-10개)
```
